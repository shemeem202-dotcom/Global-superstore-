# Global-superstore-sales-analysis-dashboard

## Project Overview

This project contains an interactive Power BI dashboard developed to analyze and visualize sales transaction data across multiple dimensions such as country, region, market, city, and shipping mode.

The dashboard helps stakeholders quickly understand sales performance, shipping trends, and regional insights through clear visuals and structured reporting.

## Objectives

Perform basic data cleaning for accurate analysis

Analyze sales performance across:

Countries

Regions

Markets

States and Cities

Visualize shipping mode distribution (percentage-wise)

Create interactive dashboards for business decision-making

## Tools & Technologies

Power BI Desktop

Data Modeling

DAX (basic measures)

Interactive Visualizations

Filters & Slicers
